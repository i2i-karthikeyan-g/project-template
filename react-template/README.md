# React.js web Application Frontend

A modern React + TypeScript application built with Vite, featuring a beautiful UI with Tailwind CSS and PrimeReact components.

## 🛠 Technologies

- **React 19** - Latest React with TypeScript
- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **PrimeReact** - Rich UI component library
- **PrimeIcons** - Icon library
- **Axios** - HTTP client with interceptors
- **React Hook Form** - Form validation and management
- **React Router v7** - Client-side routing

## 🎨 Design System

### Custom Color Palette
- **Primary**: `#003366` - Used for buttons, highlights, and brand elements
- **Background**: `#ffffff` - Clean white backgrounds
- **Text**: `#030200` - Dark text for optimal readability


## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`


### Build for Production
```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.

### Environment Variables
Create a `.env` file for environment-specific configuration:
```env
VITE_API_BASE_URL=https://your-api-url.com/api
```
